Research direction

Source signals in short news text.

I am shaping a research direction around short public-facing text: what signals models recover, which patterns are robust, and how evaluation separates language from dataset artifacts.

Current artifact: a Fox/NBC headline classifier with documented limitations and planned split, baseline, and error-analysis checks.

Agenda

Questions, evidence, and next experiments

Source-associated signals in short text

What source-associated textual signals are recoverable from headlines without using article bodies or explicit metadata?

Current evidence
News Source Classification built a 300k-headline Fox/NBC dataset and reached 87% accuracy with BERT-based representations.
Planned evaluation
Text normalization, embedding-based classifiers, baseline comparison, and controlled dataset splits.
Next experiment
Add topic-controlled or time-split evaluation to test whether the model learns source-associated language rather than event distribution.

Evaluation beyond a single accuracy number

Which errors reveal genuine ambiguity, and which errors point to leakage, topic imbalance, or scraping artifacts?

Current evidence
The current news classifier reports accuracy and baseline lift; the next useful evidence is error analysis and per-class behavior.
Planned evaluation
Confusion analysis, error buckets, baseline definitions, and checks for source/time/topic imbalance.
Next experiment
Write a research appendix for the news classifier with dataset split details, baseline names, and representative mistakes.

Data systems for repeated analysis

How can analytics systems expose comparison context and uncertainty without making the interface harder to use?

Current evidence
HoopsIQ organized fragmented NBA data into search, dashboards, trends, and shot heatmaps for repeated analysis tasks.
Planned evaluation
SQL-backed data modeling, API design, dashboard workflows, and task-oriented visualization.
Next experiment
Document the data model and user workflows more clearly, then connect the work to decision-support interface design.
